# Sholly Academy - E-commerce Platform

A comprehensive e-commerce website built with modern web technologies.

## Project Overview

This project is an e-commerce platform designed to provide a seamless shopping experience with features including:
- Product catalog
- User authentication
- Shopping cart
- Order management
- Payment processing
- Admin dashboard
- To-Do List Application

## Getting Started

See individual feature documentation for setup instructions.

## Features

### E-commerce
- Product catalog
- Shopping cart
- Checkout process
- Order tracking

### To-Do List App
- Add, edit, delete tasks
- Local storage persistence
- Mark tasks complete/incomplete
